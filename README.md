# Ravian Matrix Systems - Corporate Website

This is a Next.js App Router project for Ravian Matrix Systems, a premium GIS / Remote Sensing / IT services company.

## Tech Stack
- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- GSAP & React-GSAP
- shadcn/ui & Radix concepts

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Architecture

- `app/`: Next.js pages and layouts (Home, About, Services, Solutions, Insights, Contact)
- `components/site/`: Global components like Navbar and Footer
- `components/sections/`: Page sections like Hero, CoreServices, CTASection
- `components/ui-custom/`: Custom UI components like GeoDashboardVisual, PageHero, ContactForm
- `lib/data.ts`: Centralized content data for the entire website

## Design Principles
- Minimalist and structured aesthetic
- Warm off-white backgrounds with mustard accents
- Technical typography (Inter & IBM Plex Mono)
- "Signal Grid" visual language with subtle geospatial cues
